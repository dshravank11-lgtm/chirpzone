# **App Name**: ChirpZone

## Core Features:

- Account Creation: User authentication via email or Google.
- User Profile: Personal profile page with photo, bio, and post history.
- Anonymous or Named Posts: Create and post text-based updates; choose to post anonymously.
- Feed Interaction: View, like, comment on, and share posts from other users.
- Friend Connections: Add friends via username or friend code.
- Real-Time Chat: Send real-time text messages with typing indicators.
- Friend Request Notifications: Request to connect to other users; receive and manage friend requests.

## Style Guidelines:

- Primary color: Warm orange (#FFA500) to create an inviting feel.
- Background color: Off-white (#FAF9F6), slightly desaturated, for a clean look.
- Accent color: Vibrant green (#90EE90), to highlight important UI elements.
- Font: 'PT Sans' (sans-serif), a humanist font for both headings and body text, ensuring legibility and a friendly appearance.
- Use a soft, rounded UI design with no sharp edges to appear friendly and modern.
- Use simple, intuitive icons to help make the app easier to use and understand at a glance.
- Integrate subtle, playful micro-animations (e.g. a heart filling when a post is liked) to provide visual feedback and add a sense of fun.
